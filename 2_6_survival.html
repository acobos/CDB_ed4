Survival analysis

Variable: \(\quad\) time to an event (T), e.g., death, disease progresion, etc.

Problem: \(\quad\) censored observations (right-censoring)

  • loss to follow-up

  • no event by end of fixed study period

Goals:

  • describe survival as a function of time: \(\quad\) Kaplan-Meier method

  • compare survival in different sets of patients: \(\quad\) log-rank test

  • model survival as a function of covariates (age, sex, treatment, etc.):
    \(\quad\) Cox proportional hazards model

Survival and hazard functions

survival function: probability of surviving at least until time \(t\) \(\qquad \small{S(t) = \text{Prob}(T > t) = 1- F(t)}\)

hazard function: instantaneous death / event rate at time \(t\) \(\qquad h(t) = \frac{f(t)}{S(t)} = - \frac{d \log S(t)}{d t}\)

cumulative hazard function: accumulated risk until time \(t\) \(\quad \quad \quad \small{H(t) = - \log S(t)}\)

\(h(t)\) is conditional on not having experienced the event before t. Its not a probability.
\(H(t)\) does not have an easy interpretation.

Survival objects

The Surv() function creates suvival objects combining time and censoring/event information.

You need to specify two arguments:

  • time: \(\qquad\) the follow-up time

  • event \(\qquad\) the event/censoring indicator (see ?Surv for allowed event codings)


The result is a value for each subject: the time followed by + if censored

surv_obj <- Surv(time = d$time, event = d$status)              # creates a survival object
                                                               
head(surv_obj, 20)                                             # prints first 20 values                    
 [1]  306   455  1010+  210   883  1022+  310   361   218   166   170   654 
[13]  728    71   567   144   613   707    61    88

Kaplan-Meier (KM) curve

The Survfit function estimates survival using KM method. You need to specify a formula with a survival object (before the tilde), and either 1 (for a single curve) or a factor (for a curve for each level).

km <- survfit(Surv(time = time, event = status) ~ 1, data = d)              # KM estimation
library(survminer)
ggsurvplot(km, xlab = "Days", legend = "none",                              # plots  survfit objects
           censor=FALSE, surv.median.line = "hv")

See ?ggsurvplot for more customization options

Log-rank test

To compare survival curves with this test, use function survdiff() on a formula, with the factor defining groups after the tilde.

survdiff(Surv(time, status) ~ sex, data = d)      # note formula
Call:
survdiff(formula = Surv(time, status) ~ sex, data = d)

             N Observed Expected (O-E)^2/E (O-E)^2/V
sex=male   138      112     91.6      4.55      10.3
sex=female  90       53     73.4      5.68      10.3

 Chisq= 10.3  on 1 degrees of freedom, p= 0.001

This test can be used if:

  • Groups are independent (different subjects)

  • With similar, non-informative censoring pattern, and

  • Proportional hazards \(\implies\) curves do not cross

Survival curves with log-rank test

ggsurvplot(survfit(Surv(time, status) ~ sex, data = lung),
           pval = TRUE,                                     # plots the p-value
           pval.method = TRUE)                              # plots the method (test)

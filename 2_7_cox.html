The Cox proportional hazards model (CPHM)

\(h(t) \quad = \quad h_0(t) \quad e\ ^{\ \beta_1 x_1 + \ \beta_2 x_2 + ...+ \ \beta_k x_k} \quad = \quad h_0(t) \quad e\ ^{\beta_1 x_1} \ e\ ^{\beta_2 x_2} \ ... \ e\ ^{\beta_k x_k}\)

Models the hazard function \(h(t) = \frac{f(t)}{S(t)} = - \frac{d \log S(t)}{d t}\), the instantaneous hazard of event at time \(t\)

\(h_0(t)\) is the baseline hazard (i.e., when all \(x\)s are 0)

\(e\ ^{\beta}\) are hazard ratios; suppose \(x_1\) is an indicator (i.e., 0 or 1); then

  • \(h(t|x_1=1) \quad = \quad h_0(t) \quad e\ ^{\beta_1} \quad e\ ^{\beta_2 x_2} \quad ... \quad e\ ^{\beta_k x_k}\)

  • \(h(t|x_1=0) \quad = \quad h_0(t) \quad 1 \qquad e\ ^{\beta_2 x_2} \quad ... \quad e\ ^{\beta_k x_k}\)

So that

  • \(\frac{h(t|x_1=1)}{h(t|x_1=0)} = e\ ^{\beta_1}\)

So, it is implicitely assumed that the hazard ratio for any time t is a constant, which does not depend on \(t\)

Fitting a CPH model

Function coxph() fits CPH models

fit <- coxph(Surv(time, status == "dead") ~ sex, data = d)

Assessing the PH assumption

A test for deviation from proportional hazards can be done with function cox.zph

cox.zph(fit)
        chisq df    p
age     0.188  1 0.66
male    2.305  1 0.13
ph.ecog 2.054  1 0.15
GLOBAL  4.464  3 0.22

A significant p-value means evidence of deviation from proportional hazards.

Fore more thorough ways of assessing assumptions of the Cox PH model, see this

When PH do not hold, Accelerated Failure Time (AFT) models may be used.
